FOR RUNNING FERMIPY

1) activate conda fermipy_jan2023
2) Make sure your run directory has:
	config.yaml
	run.py
	some ltcube*.fits
	proper links to data and SC files
	proper links to galactic/diffuse/iso files
3) outputs of code that will end up in run dir
	bexpmap*
	ccube*.fits
	gt*.par
	srcmap_00.fits
	srcmdl_00.xml
	sourcefind*
